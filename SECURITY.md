# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in this project, please report it privately rather than opening a public issue.

Email: **security@auron-labs.org**

Include:

- A description of the vulnerability and its impact
- Steps to reproduce or a proof of concept
- Affected versions or packages

You should receive an initial response within 48 hours. Please do not disclose the vulnerability publicly until a fix has been released.

## Scope

This policy covers all packages in this repository under `packages/*`.
